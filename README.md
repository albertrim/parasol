# Parasol v2 — Live Beach

부산 해운대 비치 파라솔용 모바일 QR 메뉴판 / 주문 / 배달 서비스 프로토타입.

**컨셉**: "시간·날씨가 주문을 만든다"
체감온도 기반 다이나믹 부스팅, 옆 파라솔 공동구매, 노을 카운트다운, 소나기 5분 전 묶음주문, 만조 알림.

## 목업

`mockup/index.html` — 단일 HTML 파일. 외부 의존성은 Google Fonts (Pretendard Variable) 하나.

- 3개 테마 (A: 선그라스 미니멀 / B: 딥 워터 / C: 민트 페이퍼)
- 8개 시간순 시나리오 (아침 → 평소 → 만조 → 더위시작 → 폭염 → 소나기 → 노을 → 야간)
- 3개 역할 × 5단계 풀 스토리 시뮬레이션 (손님 / 사장님 / 배달원)

로컬에서 보려면 그냥 `mockup/index.html`을 브라우저로 열면 됩니다.

## 배포

Vercel 정적 호스팅. 루트의 `vercel.json`이 `mockup/` 폴더를 사이트 루트로 매핑합니다.
